CaffeineTracker je widget pro Garmin hodinky, který ti pomůže mít přehled o příjmu kofeinu a nezasahovat si do spánku.

Koupit unlock Otevřít v Connect IQ Store

Po zaplacení ti přijde unlock kód automaticky e-mailem. Na checkoutu vyplníš svůj App code.

Jak to funguje

  • Nainstaluješ widget z Connect IQ Store
  • V aplikaci si zobrazíš App code
  • Klikneš na Koupit unlock a na Stripe checkoutu vložíš App code
  • Po úspěšné platbě ti přijde e-mailem unlock kód

FAQ

Kde najdu App code?
V aplikaci v sekci Unlock / License (doplň dle UI).

Kdy přijde unlock kód?
Typicky do pár minut po platbě (automatizace přes webhook).

Nechci zadávat App code do Stripe
Můžeš použít kontakt/support a poslat ho ručně – ale automatika je rychlejší.
